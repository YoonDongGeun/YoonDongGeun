## What if

- What if를 추구하는 개발자.
  
    처음에는 How만을 보며 어떻게 사용하는지만을 배웠다면, 
    
    이제는 Why를 생각하여 왜 이 기술을 사용하는지 생각합니다.
    
    더 나아가 What if 를 생각하려합니다. 당장 이 기술을 왜 사용하는지도 중요하지만, 나중에 추가될 서비스와 유지 보수도 생각하는 개발자가 되고 있습니다.
